The `if_else` function is useful for changing certain elements in a data frame.

``` r
if_else(letters == "h", "H", letters)
temp <- gapminder %>%
  mutate(lifeExp = if_else(country == "Canada" & year == 1952, 70, lifeExp)) %>% 
  filter(country == "Canada")
```
